Sarvh.in — Admin (GitHub-only)

This admin page uses GitHub's own editor and pull request workflow. No external services or OAuth apps are required beyond your GitHub account.

Quick actions

  • Home (index.html): Edit on GitHub
  • Create a new page: Create new page
  • Upload or add an asset: Create assets/uploads/
  • Open repository (all files): Repository

How to edit

  1. Click an "Edit on GitHub" button. You'll be taken to GitHub's web editor. Sign in with your GitHub account (your GitHub username and password if not already signed in).
  2. Make changes in the editor. When finished click Propose changes. This will create a branch and open a Pull Request.
  3. Review the Pull Request and merge it to publish changes to the main branch. If GitHub Pages is enabled, the site will update after merge.

Notes & tips

  • This page intentionally does not require any OAuth app or external JS. It only uses links to GitHub's web interface.
  • If a file does not exist yet, use the "Create new page" or "Create assets/uploads" links to add it.
  • Uploading images: create the folder assets/uploads via the "Create assets/uploads" link, then edit that folder and use "Add file" > "Upload files" in GitHub's UI.
  • Back up important files before large edits. Editing index.html will replace the file contents.
